# План: Интерактивная оргструктура «Шугнов Голд»

Одностраничное React-приложение с переключателем между двумя видами оргструктуры компании полного цикла промывки золота.

## Основные экраны

**1. Шапка**
- Название компании, краткое описание (2000 га, 8× ППМ-5, р. Шугнов, Ховалинг, Таджикистан)
- Тоггл-переключатель видов: «🗺 По локациям» / «⚙ По производственному циклу»

**2. Вид «По локациям» (физическое размещение)**
- Стилизованная карта-схема с 4 зонами:
  - Локация 1 — Центральный штаб (плато)
  - Локация 2 — ШОУ (бункер)
  - Локация 3 — Промзона и логистика
  - Локация 4 — Участки добычи (Нижнее и Верхнее течение реки)
- Река отрисована SVG-полилинией через всю схему
- Каждая локация — кликабельная карточка-зона; клик раскрывает панель со списком отделов и должностей внутри
- Цветовое кодирование зон + иконки (Lucide: Building2, ShieldCheck, Wrench, Waves, Mountain)

**3. Вид «По производственному циклу»**
- Горизонтальная цепочка этапов (Sankey-style / connected pipeline):
  1. Разведка (Геология/маркшейдерия)
  2. Подготовка (Вскрыша)
  3. Добыча песков (Экскаваторы/бульдозеры)
  4. Транспортировка
  5. Промывка (8× ППМ-5 → 4 участка)
  6. Доводка шлиха (ШОУ)
  7. Плавка / Аффинаж
  8. Учёт и сбыт
- Параллельные «поддерживающие» дорожки снизу: Механика, Энергетика/Вода, Снабжение, Безопасность, Экология, HR/Финансы
- Стрелки с анимацией потока (CSS), на узле — счётчик персонала и кнопка «детали»
- Клик по этапу → боковая панель со списком должностей и связью с физической локацией (бейдж «Локация 2 — ШОУ»)

**4. Общая боковая панель (Drawer)**
- Заголовок отдела, физическая локация, перечень должностей с количеством
- Кнопка «Показать на другом виде» — переключает режим и подсвечивает соответствующий узел

**5. Лёгкая статистика снизу**
- Всего сотрудников, машин ППМ-5, участков, площадь

## Технические детали

- React + TypeScript + Tailwind + shadcn (Sheet, Card, Tabs/ToggleGroup, Badge)
- Единый источник данных `src/data/orgStructure.ts`: массив отделов, каждый имеет `id`, `name`, `locationId`, `cycleStageId`, `positions: {title, count}[]`, `icon`, `description`
- Два компонента визуализации читают один датасет: `LocationView.tsx` (абсолютное позиционирование зон на SVG-фоне с рекой) и `CycleView.tsx` (горизонтальный flow со SVG-коннекторами)
- Общий `DepartmentDrawer.tsx` (shadcn Sheet) с кнопкой кросс-навигации между видами
- Состояние выбранного вида и отдела — в `Index.tsx` через `useState`; URL-параметры `?view=...&dept=...` для шаринга
- Адаптив: на мобильном вид «по локациям» становится вертикальным стеком зон; цикл — горизонтальный скролл
- Дизайн: тёмная земляная палитра (тёмно-зелёный/охра/золото) под золотодобычу, шрифт Inter
- Без backend — все данные статичны
- SEO: title «Оргструктура — Шугнов Голд», мета-описание, H1
